## Tutorial para testes do banco de dados usando o localhost

O arquivo apra a inserção doa dados reais deve ser o código e ser rodado no PgAdmin, e as planilhas EVENTOS e METADADOS.
As planilhas devem estar em uma pasta que o PostgreSQL reconheça, minha recomendação é a pasta "data", dentro da pasta do PostgeSQL do seu computador.
Com os arquivos na pasta, e o script pronto para rodar, só executar ele e as tabelas serão criadas com os dados.
